# CSS

Ficheiros na raiz de `css/` (sem subpastas) para caminhos simples nos HTML:

| Ficheiro | Uso |
|----------|-----|
| `global.css` | Reset, cabeçalho, rodapé, botões, ajuda (`?`), toasts |
| `forms.css` | Formulários e campos |
| `landing.css` | Hero, secções da página inicial |
| `calendar.css` | Grade semanal de rodas |
| `roda-card.css` | Cards de roda (modal, compacto) |
| `cards.css` | Cartões genéricos (notícias, etc.) |
| `dashboard.css` | Layout geral dos painéis |
| `dashboard-shell.css` | Sidebar, tabelas, KPIs dos dashboards |

Ordem típica: `global.css` primeiro; depois estilos específicos da página.
